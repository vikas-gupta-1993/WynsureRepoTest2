General Ledger

In Wynsure, General Ledger is treated as an independent module.
For each operation done on an account, a transaction containing all the created lines is sent to the General Ledger module. The module then treats these lines to create the GL entries using the GL entry types set in the application.
For each line, a processing type is selected. Using this processing type, the billing type and the statutory reporting line, the matching entry types are selected. For each of these entry types, a General Ledger entry is created.

General Ledger entry types

The General Ledger entry types define how the General Ledger entries will be built. It defines two general ledger accounts that will be impacted by the transaction; one for credits, the other for debits.

Entries creation

The entries are created from the information in the entry types and in the line. The following information is set from the entry type:

·         Credit account,

·         Debit account,

·         Processing type,

·         Billing type.

The fields filled with the line information are:

·         Amount,

·         Account line,

·         Account transaction.

The journal date is the date where the line has been created. The other parameters of the entries are created using the parameters described bellow.

Bill generation

Subscribed service billing

When billing a subscribed service, the following parameters are used to select the entry types:

·         Processing type: Adjust increase receivable due and Adjust decrease receivable due for adjustment, Create receivable due or Cancel receivable due otherwise.

·         Line of business of the service.

·         Lock box,

·         Issuing company.

 

The information of the General Ledger entry is set as follows:

·         Invoice number of the bill,

·         Contract number,

·         Issuing company,

·         Line off business and associated statutory reporting line, product and product Id of the service,

·         First year or renewal status of the service,

·         Lock box of the bill.

Waiver of premium billing

When billing a waiver of premium, the information is set as it is done for a subscribed service but with the processing types: Create waiver of premium and Reverse waiver of premium.

 

Fee billing

When billing a fee, the following parameters are used to select the entry types:

·         Processing type: Create fee due or Cancel fee due.

·         Lock box,

·         Issuing company.

The information of the General Ledger entry is set as follows:

·         Invoice number of the bill,

·         Contract number,

·         Issuing company,

·         Lock box of the bill.

Payment creation

When creating or cancelling a payment, the following parameters are used to select the entry types:

·         Processing type: Create Payment or Cancel Payment,

·         Kind of lock-box: Wholesale or Retail,

·         Issuing company.

 

The information of the General Ledger entry is set as follows:

·         Reason code: Receipt, Conditional Receipt, Unallocated Payment depending on the account to which the payment is attached: premium account, suspense account, global suspense account.

The following information is set from the payment:

·         Issuing company,

·         Lock-box,

·         Lock-box kind,

·         Payer,

·         Bank receive date,

·         Journal post date,

·         Batch Id.

Reconciliation

When the reconciliation is done, the following parameters are used to select the entry types:

·         Line of business of the service.

·         Processing type: Apply payment due or Unapply payment due,

·         Lock box,

·         Issuing company.

 

The information of the General Ledger entry is set as follows:

·         Invoice number of the bill,

·         Contract number,

·         Issuing company,

·         Line off business and associated statutory reporting line, product and product Id of the service,

·         First year or renewal status of the service,

·         Lock box of the bill.

Bill or bill line write off

For a write off, the General Ledger entry is created as it is done for reconciliation but with the processing types: Create write off / Reverse write off.

Compensation calculation

When the compensation is calculated, the entry types are selected using:

·         Processing type: Create pay commission or Cancel pay commission,

·         First year or renewal status of the compensated service,

·         Issuing company of the compensation contract,

·         Line of business of the compensated service.

The information of the entry is set using:

·         Contract number,

·         Statement number,

·         Issuing company,

·         Line off business and associated statutory reporting line, product and product Id of the compensated service.

 

 

 

Move money from / to General Ledger

When moving money from or to General Ledger, the following parameters are used to select the entry types:

·         Processing type: Move money from cash or Move money to cash depending on the type of move,

·         Reason code,

·         Used in move money,

·         Issuing company.
