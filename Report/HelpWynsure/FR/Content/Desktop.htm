Desktop

Definition

By launching Wynsure, a log in window will appear to identify the user. According to the user identification, which lies to defined authorization domain, it shows desktops with different entry points.

 

Wynsure contains the entry points as following:

  • My Work: shows tasks to be done and proposal done
  • Products: contains available insurance products, new products can be defined here.
  • Actors: records of actors involved.
  • Contracts: records of personalised contracts.
  • Claim: records of claims and relative information, for claim declarations also,.
  • Premium management:
  • Compensation: records of involved roles and for compensation plans and contracts definition
  • Reporting tools
  • Rule engine
  • Administration: for user & security management and modules’ settings
  • Health care
  • Investments
  • Reinsurance
  • Batches: batch process setting
  • Test
  • Quote: creation of customized quotes

Main

 

Take Super manager log in as example, on the left of the screen, we can find all entry points authorized to the current user. These entry points, shown in a tree structure, cover different insurance functions. These entry points are also available in the Actions menu bar. On the left bottom corner of the screen, you can see the identification of the user logged in. Help Pages can be triggered by using the help icon on the right top corner.
