Définition

La personne morale représente un entité qui est reconnu par létat, une entreprise par exemple.

Une personne morale est défini par les paramètres suivants:

·                         Information principale (Adresse légale, )

·                         Autres coordonnées

·                         Rôles: Une personne morale peut être client, distributeur, courtier, prestataire de service, administrateur tiers, filiale ou assureur.  (Différents informations sont définies pour chaque rôle.)

·                         Comptes bancaires

·                         Relations: Sa relation avec dautre personnes ou personnes morales.

·                         Membres: employées qui travaillent pour la personne morale

·                         Contacts

·                         Hiérarchie: Où on peut définir le parent de cette personne morale.

Utilisations

Dans Wynsure, une personne morale est utilisée dès que la description de la société est nécessaire:

·                                 Les sociétés qui utilisent Wynsure

·                                 Les sociétés qui vendent le produit (Distributeurs)

·                                 Les sociétés qui touchent la commission en faisant signer le contrat (Courtier)

·                                 Les sociétés qui est ladministrateur tiers

·                                 Les sociétés qui achètent  lassurance (Client qui signe le contrat assurance)

·                                 Lentité qui offre certain service à travers les produits du distributeur (Prestataire de service, Prestataire de soins)

·                                 Les sociétés qui assument le risque de  contrat assurance (Assureur)

·                                 Les sociétés qui partagent le risque de certains contrats assurances (Réassureur)

·                                 Société qui permet ses employées lengagement de produit collectif dune autre société(Filiale)

Classification

Classification est appliqué aux personnes définies dans Wynsure

Si la classification est définie, lutilisateur doit spécifier les valeurs des classifications pour chaque création du membre.

Ecran Principal

Lécran principal affiche la raison sociale, le nom court et le numéro interne de la personne morale.

 

Lécran est divisé en plusieurs onglets:

o        Principal

o        Autres coordonnées

o        Rôles

o        Comptes Bancaires

o        Relations

o        Membres

o        Contacts

o        Hiérarchie

o        Autre

Onglet Principal

Longlet principal ne contiens que ladresse de la société (Domiciliation).

 

Onglet Autres Coordonnées

 

Longlet affiche dautres coordonnées appartient à la personne moral, vous avez les choix de 3 types de coordonnée, ladresse postal, le numéro de téléphone et ladresse email.

Onglet Rôles

 

Ceci présente les différents rôles que la personne morale peut jouer : Client, Distributeur, Administrateur tiers, Filiale, Courtier, Prestataire de service, Prestataire de soins, Assureur, etc.

Client

§         Onglet Distributeurs

 

Le numéro client identifie le client. Un client doit déclarer son distributeur afin de pouvoir sinscrire au contrat collectif avec ce distributeur.

 

§         Onglet Gestion

 

Web Login contient le nom et le mot de passe pour se connecter par internet en tant que client. Paramètres de gestion contient 1) Paramètres du workflow pour le client, et 2)  Domaine dautorisation, qui précise la domaine dautorisation liée. ( Chaque personne morale peut être liée à une domaine dautorisation. La domaine dautorisation est un élément qui décrit lorganisation de la société, présentée par la fenêtre ci-dessous.) Contacts internes consiste le client contact information dans certaine période.

Distributeur

 

Le Distributeur est une société qui vend les produits assurances et qui offre le contrat assurance. Le contrat est signé entre le client et le distributeur.

 

Quand un produit est défini dans Wynsure, le produit doit être lié à un distributeur. Wynsure permet plusieurs distributeurs dans le même système dinformation. Chaque produit, contrat et entité est lié à un des distributeurs. Ceci permet la société de gérer ses produits /Contrats séparément selon sa structure géographique. Ainsi, la société peut gérer les produits /Contrats dune autre société qui na pas de système dinformation.

 

§         Onglet Principal

 

Dans longlet principal, vous pouvez définir le format du générateur de numéro de contrat / contrat de compensation/ facture/décompte, etc.

 

§         Onglet Paramétrage

Ceci contient les devises possible chez le distributeur et ses méthodes de paiement possibles. 

 

§         Onglet Comptes

Wynsure génère les comptes dattente globaux pour le distributeur en tant que le compte défaut.

Administrateur Tiers

Un administrateur tiers est une société indépendante confiée par un groupe à administrer son plan dassurance. Ladministrateur tiers juge le sinistre et décide la possibilité de remboursement.

 

Vous pouvez choisir un administrateur tiers pour une certaine période.

Filiale

La filiale est une société dans laquelle ses employées peuvent engager le contrat collectif dune autre société. Par exemple, on suppose que AOO est une filiale de la société NIKE, donc les employées chez AOO peuvent souscrire au contrat collectif de Nike.

 

Une personne morale peut être la filiale de plusieurs sociétés pour une période définie.

 

Courtier

Le courtier est une personne ou personne morale qui négocie les règles assurances avec lassureur au nom du client. Il peut recevoir une commission de lassureur pour le travail quil a réalisé. Il est aussi lentité qui signe le contrat commissionnement.

 

Un courtier contient les informations complémentaires suivantes :

·                                            Numéro courtier

·                                            Web Log in

·                                            Etc.

 

Prestataire de Services

Le prestataire de services est ce qui offre le service non-assurance à travers le produit du distributeur.  Vous pouvez lui donner une description.

Prestataire de soins

 

Le prestataire de soins peut être un médecin, un hôpital, un infirmier, etc., qui intervient en cas de sinistre.

Un numéro intervenant est défini à identifier le prestataire de soins.

 

§         Onglet Principal

Vous pouvez retenir des informations complémentaires à propos du prestataire de soins, genre le type dintervenant et ses spécialités.

 

§         Onglet Spécifications

Ceci contient les informations détaillées sur le prestataire de soins, y compris UPIN, Code postal, etc.

 

§         Onglet Description

Vous pouvez ajouter une description à propos du prestataire de soins.

 

§         Onglet Structure

Ceci nest que pour le cas que le prestataire de soin est une personne morale. Si le prestataire de soins est structuré, cest-à-dire quil nest pas indépendant, Vous pouvez décrire sa structure en détail.

 

Une fenêtre va apparaître en double cliquant sur la racine de larbre, où vous pouvez définir le détail dunité de prestataire de soins.

 

Assureur

Lassureur est une société qui prend la risqué associée du produit assurance.

 

Un numéro dassureur est demandé pour identifier lassureur.

 

Comptes Bancaires

Wynsure permet la personne morale davoir plusieurs comptes bancaires.

Vous pouvez consulter ou modifier les comptes existants, comme ci-dessous ( Trois types de comte est considéré dans Wynsure, Compte US, Compte IBAN et Compte français, afin de satisfaire les cas différents.) Un compte bancaire est identifié par le nom de la banque, adresse, etc.)

 

o        Compte US

Ceci est utilisé en cas de compte bancaire aux US.

 

o         Compte IBAN

Ceci est utilisé en cas de compte bancaire international.

 

o        Compte français

Ceci est utilisé en cas de compte bancaire français.

Relations

Vous pouvez donner les relations entre une personne morale et dautre personnes ou personnes morales. Acteur 1 est la personne morale actuelle.

 

La relation entre acteurs consiste les informations suivantes : un libellé, les deux acteurs concernés et une période de validation.

Membres

Ceci affiche la liste demployées appartiennent à cette personne morale.

Contacts

Un contact contient une période de validation, une description et un identité (y compris le nom, le numéro de téléphone et ladresse). Ladresse du contact prend directement les adresses disponibles de la personne morale.

 

Dans la facturation, un contact est nécessaire afin de donner la destinataire de la facture quand il sagit dun contrat collectif.

Hiérarchie

Hiérarchie dune personne morale permet de définir lancêtre et afficher ses descendants.

Autre

Un logo peut être attaché à la personne morale, qui est affichi en haut à droite dans le scénario.

Définition

La personne physique est représentée par la personne dans Wynsure. Vu que la même personne peut intervenir dans plusieurs rôles (Courtier, Membres, Utilisateurs, etc.), ces rôles nhéritent pas de la personne mais sont liées à la personne.

Une personne peut être liée avec dautre personne, cette relation peut être mono directionnelle (être lenfant de) ou multi directionnelle (être marié)

Utilisations

·                                                                    Abonné de contrat individuel

·                                                                    Liée à un membre dune unité dorganisation

·                                                                    Liée à un utilisateur dapplication (Voir Module Autorisation)

·                                                                    Un Courtier peut être une personne (Voir Module Commissionnement)

·                                                                    Contact dun contrat

·                                                                    Destination dune facture ou dun commissionnement (Voir Module Facturation)

·                                                                    Débiteur dun paiement (Voir Module comptabilité)

·                                                                    Personne couverte dans un contrat  (Voir Module Contrat). Relation désigne les candidatures couvertes possibles.

·                                                                    Etc.

 

Classification

La classification des personnes est possible dans Wynsure.

Une fois les classifications sont définies, lutilisateur doit spécifier à quelle classification appartient le client chaque fois quand il défini un client.

Classification de personne est utilisée dans la module produit individuel pour customiser les règles.

Ecran principal

 

Lécran est divisé en plusieurs onglets:

o        Principal

o        Autres coordonnées

o        Rôles

o        Comptes Bancaires

Principal

En général, ce onglet contient les données suivantes:

o        Personne identité, y compris Civilité, nom, prénom, 2nd prénom, autre prénom, suffixe, date de naissance, etc.

o        Coordonnées principales, y compris adresse, téléphone, email, etc.

o        Information Civile, y compris SSN, Etat civil, Lieu de naissance, etc.

o        Valeur de classification

Autres Coordonnées

Vous pouvez définir dautre coordonnées de la personne, comme adresse secondaire, contact en urgence, etc.

Vous avez les choix de 3 types de coordonnée, ladresse postal, le numéro de téléphone et ladresse email.

 

Rôles

Ceci contient les informations sur les rôles quune personne peut jouer, cest-à-dire Client, Courtier et Prestataire de soins.

 

Il contient un ou plusieurs onglets selon le rôle joué. Les information de chaque onglet sont pareils que ceux dans le cas de personne morale.

Synthèses

Modèle

Utilisation

Le synthèses est disponible uniquement quand il sagit dun client.

 

Il peut être activé soit en double cliquant sur le client même ou en cliquant sur licône synthèses.

Ecran Principal

Lécran principal consiste 2 parties, une partie qui affiche la synthèse générale de cette personne/personne morale sous forme dun document, une autre partie qui affiche la synthèse spécifique de cette personne/personne morale, genre contrats souscrits, comptes clients, factures, paiements, remboursements, contacts, devis, prestation, affiliés(ceci concerne que la personne morale) and sinistres. En général, tous opérations concerne le client peut être fait ici.

 

Synthèse Générale

Dans cette partie, vous pouvez consulter la synthèse de certains onglets définis dans la personne morale/la personne, dans lordre suivant: principal, autres coordonnées, contacts, comptes bancaires et rôles. Ces informations sont présentées dans une fenêtre blanc, vous pouvez aussi les consulter dans leurs onglets originaux.

Synthèse Spécifique

Contrat

Le contrat représente tout type de contrat assurance disponible pour lindividuel et le groupe.

Cet onglet affiche tout contrats souscrits. En plus, la souscription du nouveau contrat nest fait quici.

Comptes Clients

Le compte client enregistre les transferts à propos de ce client. Par exemple, vous pouvez définir un compte facturation pour un contrat et un compte attente pour les transferts défauts.

Factures

Factures sont générées selon les règles de tarifications, un certain montant sera facturé à un moment donné.

Dans cet onglet, toutes les factures de ce client sont affichées, avec informations genre statue, numéro de facture, montant, etc.

Paiements

Une fois le client régle un paiement, il est enregistré. En général, chaque paiement corresponde à une facture.

Remboursements

Remboursement est disponible en cas de sinistre.

Contacts

Ceci enregistre le contenu et la suite de chaque contact entre le client et lutilisateur Wynsure, la description contact client est comme ce-dessous, 

Devis

Dans Wynsure, vous pouvez demander un devis avant de signer le contrat. Lutilisateur peut simuler le contrat selon ses préférences. Le devis peut être disposé ou validé. Cette fonction est aussi disponible dans le menu sur le bureau.

Prestations

Ceci affiche les prestations rendues au client.

Affiliés

Ceci liste les employées dans la société. (Il ne concerne que la personne morale)

Sinistres

Sinistre est déclaré en cas de perte couverte par lassurance.  Wynsure permet dautre personne que le client de déclarer la sinistre.

Définition

Le membre est une personne qui travaille pour une personne morale (dans une entreprise, for exemple).

Il consiste les donnés suivants :

·                                            Personne Morale

·                                            Personne

·                                            Information du salarié

·                                            Catégorie de travail

·                                            Statut de lemploi

·                                            Salaire

·                                            Heures travaillées

·                                            Etc.

Utilisation

·                                                                    Description dune personne morale ou dune unité organisationnelle

·                                                                    Inscription au contrat collectif

Ecran Principal

 

Lécran contient,

o        Nom de la personne moral, qui se réfère à la personne moral à laquelle le membre appartient. Lécran principal de cette personne morale peut être affiché.

o        Nom et date de naissance de la personne, qui se réfère à une personne dans la classe Personne, et affiche lécran principal de cette personne.

o        Information de Membre, y compris la date dembauche, Id. salarié.

o        Informations salariés

 

Adresse

Définition

 

Wyde Framework fournie un component dadresse pour ladresse française et ladresse américaine. Ladresse peut être utilisé pour les pays différents, cest pour ça que Wyde crée un component multi-pays code postal qui permet de caractériser le code postal, villes, etc.

Utilisation

Reliée à une personne, une personne moral, une unité organisationnelle, un travail, etc.

Pour linstant, Son utilisation est principalement dans la module facturation pour la distribution de la facture et dans la module commissionnement pour la distribution du relevé.

Personne Desktop

Vous pouvez chercher and afficher les personnes selon certains critères , par exemple le nom, le prénom, les rôles, etc. les options de Date de naissance et de Sexe doit être crochées pour les activer.

Vous pouvez chercher and afficher les personnes morales selon certains critères , par exemple le raison sociale, le nom court, les rôles, etc. Vous pouvez créer une nouvelle personne morale en donnant les informations détaillées. (Voir Personne Morale)

Client Desktop

Tout acteurs qui jouent la rôle client est considérés comme un client. Un client peut être une personne ou une personne morale.  La création du profil client est fait pendant la création de lacteur, vous pouvez seulement faire une recherche et afficher les clients ici. Cependant, les critères de recherche sont différents pour la personne et la personne morale.

 

Client en tant quune personne morale est identifié par son nom et son nom court.

Client en tant quune personne est identifié par son nom et son prénom.
